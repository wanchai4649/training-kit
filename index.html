ยินดีต้อนรับสู่ GitHub หน้า เครื่องกำเนิดไฟฟ้าอัตโนมัติหน้านี้เป็นวิธีที่ง่ายที่สุดในการสร้างหน้าเว็บที่สวยงามสำหรับทุกโครงการของคุณ ผู้เขียนเนื้อหาของหน้าเว็บของคุณที่นี่ ใช้ GitHub รส Markdown เลือกแม่แบบที่สร้างขึ้นโดยนักออกแบบและเผยแพร่ หลังจากที่หน้าของคุณจะสร้างคุณสามารถตรวจสอบสาขาใหม่ GH-pages ในประเทศ หากคุณกำลังใช้ GitHub สก์ท็อปเพียงแค่ซิงค์พื้นที่เก็บข้อมูลของคุณและคุณจะเห็นสาขาใหม่ ### ออกแบบแม่แบบเราได้สร้างขึ้นมาบางแม่แบบหล่อเพื่อให้คุณใช้ ไปข้างหน้าและคลิก 'ไปยังรูปแบบ' เพื่อเรียกดูผ่านพวกเขา คุณสามารถกลับไปแก้ไขหน้าเว็บของคุณก่อนที่จะเผยแพร่ หลังจากประกาศหน้าของคุณคุณสามารถทบทวนกำเนิดหน้าและเปลี่ยนไปใช้รูปแบบอื่น เนื้อหาบนหน้าเว็บของคุณจะถูกเก็บรักษาไว้ ### การสร้างเพจด้วยตนเองหากคุณต้องการที่จะไม่ใช้เครื่องกำเนิดไฟฟ้าอัตโนมัติผลักดันสาขาที่ชื่อ GH-pages พื้นที่เก็บข้อมูลของคุณเพื่อสร้างหน้าด้วยตนเอง นอกจากการสนับสนุนเนื้อหา HTML ปกติหน้า GitHub สนับสนุน Jekyll, ง่าย, บล็อกกำเนิดเว็บไซต์ตระหนักคงที่ Jekyll ทำให้ง่ายต่อการสร้างส่วนหัวทั้งไซต์และท้ายกระดาษโดยไม่ต้องคัดลอกไปทั่วทุกหน้า นอกจากนี้ยังมีการสนับสนุนบล็อกที่ชาญฉลาดและคุณสมบัติอื่น ๆ แม่แบบขั้นสูง ### ผู้เขียนและผู้ร่วมสมทบคุณสามารถเอ่ย @ ชื่อผู้ใช้ GitHub การสร้างลิงค์ไปยังโปรไฟล์ของพวกเขา ส่งผลให้ `` องค์ประกอบจะเชื่อมโยงไปยังผู้มีส่วนร่วมของโปรไฟล์ GitHub ตัวอย่างเช่นในปี 2007 คริส Wanstrath (@defunkt), PJ Hyett (@pjhyett) และทอมเวอร์เนอร์เพรสตัน (@mojombo) ก่อตั้งขึ้น GitHub ### การสนับสนุนหรือการติดต่อมีปัญหากับหน้า? ตรวจสอบ [เอกสาร] เรา (https://help.github.com/pages) หรือ ติดต่อฝ่ายสนับสนุน และเราจะช่วยให้คุณจัดเรียงออก
